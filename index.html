ENGL1167 Creative Final Project

Which Spider-Man Are You?

This project treats Spider-Man not as one fixed character, but as a figure shaped by adaptation, citation, and collaboration. By turning the differences between three live-action film portrayals into a quiz, the website shows how each version emphasizes different traits, tones, and narrative priorities.

The quiz format turns interpretation into an interactive classification system. Each choice translates an idea about tone, relationships, heroism, mentorship, or identity into points for one version of Spider-Man.

This project focuses only on the three major live-action Spider-Man film portrayals in order to keep the scope narrow and compare three adaptations working within a similar cinematic format.

Project Explanation

The quiz format turns interpretation into interaction. Rather than arguing for one definitive Spider-Man, the project asks users to move through a set of choices that reflect different authorial priorities. The questions are based on recurring differences across the three live-action portrayals: moral responsibility in the Raimi films, emotional identity in the Webb films, and mentorship/collaboration in the MCU films. The project focuses on live-action Spider-Man rather than animated Spider-Verse adaptations so the comparison stays within a shared cinematic format. Each answer connects to tone, characterization, relationships, or narrative priorities, and the final result shows which pattern of emphasis appears most often.
